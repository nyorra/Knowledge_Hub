# AI_Knowledge_Hub

Web-приложение, позволяющее удобно получать информацию на основе ваших файлов из внутреннего хранилища с помощью ИИ.

## Стек
- Backend: Python, FastAPI
- Frontend: React
- IDE: PyCharm (backend), VSCode (frontend)

---

## Установка и запуск

### 1. Backend
1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/nyorra/AI_Knowledge_Hub.git
   cd AI_Knowledge_Hub/backend

2. Установите зависимости:
    ```bash
    pip install -r requirements.txt

3. Запуск приложения
    ```bash
    python main.py

---

### 2. Frontend
1. Перейдите в папку frontend`а:
    ```bash
    cd ../frontend

2. Установка зависимостей:
    ```bash
    npm install

3. Запуск react приложения:
    ```bash
    npm install
